## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open your browser to `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```

## Pages

1. **Home** - Hero section with book cover, features, author preview, and call-to-action
2. **About the Book** - Detailed information about "Greatness Through Persecution"
3. **About Will** - Author biography, podcast information, and values
4. **Book Tour** - Upcoming events, virtual sessions, and booking information
